Project Description:

This project is a simple dashboard that allows users to create task lists, notes, and reminders to view on a calendar.

How-to-use Section:

To-do list board:

To use the To-do list board make a new list by pressing the button.
This list can be deleted at anytime by pressing the button created on the list.
A task can be added onto the list by pressing the button created on the list.
Each task will also have its own delete button. As well as a check box that is red for unfinished and green for finished, this can be toggled by clicking on it.
Both tasks and task lists will have a text input for a name/description this can be changed by double clicking and will be uneditable after the cursor hovers off of the input.
The user can have 5 task lists with each task list holding 14 tasks.

Note board

To use the Note board make a new note by pressing the button.
Each note has its own delete button to remove it at any time.
A note contains a text input that can hold three lines of text.
The user can have 7 notes maximum.

Calendar

The Calendar is a simple 12 month calendar display. The current date is displayed on the header under the current month.
The month can be decremented or incremented with the previous and next buttons, respectively.
Days from the previous or next month will have a gray background while days in the current month have a white background with the except of the current day having a green background.
The calendars use is connected to the Reminder board.

Reminder board

The Reminder board holds date reminders for the calendar.
A new reminder can be created by pressing the button.
Each reminder contains a color input ,
a text input ,
a date input ,
and a delete button so that it can be deleted at anytime.
Once a reminder is given a date it will be added to the calendar.
When a reminder is deleted from this board it is automatically removed from the calendar.
Reminders are color coded so they can be displayed on its date on the calendar.
The description for a reminder will show when the user hovers over the color in the calendar.
Attributes for the reminder can be changed at any time and its changes will be reflected on the calendar.
The user can have 11 reminders maximum.
